# ⚡ CPS Tracker (Click Speed Tester)

## 📌 Overview
CPS Tracker is a simple and modern desktop tool that measures how fast you can click. It shows your **live Clicks Per Second (CPS)** and tracks your **highest peak CPS** in real time.

Perfect for:
- Minecraft PvP practice 🎮  
- Click speed training 🖱️  
- Fun competitions with friends ⚡  


## ✨ Features
- 📊 Live CPS counter (updates in real time)
- 🏆 Peak CPS tracking
- 🖱️ Click anywhere in the window to register clicks
- 🎨 Modern dark-themed UI
- 🔄 Reset peak button
- ⚡ Smooth and lightweight performance

## 🛠️ How It Works
- The program tracks every click you make
- It counts how many clicks happen in the last 1 second
- That value becomes your **CPS**
- The highest CPS reached is saved as **Peak CPS**
